# Hiiragi Core

![GitHub License](https://img.shields.io/github/license/hiiragi283/hiiragi-core?style=for-the-badge)

## About

- A library mod for Hiiragi Series
- Supported mod loader: NeoForge
- Supported mc version: MC1.21.1
